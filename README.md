# Simple 2048 Game

This repository contains a simple 2048 implementation using Python and Tkinter.

## Features

- Local high score is saved to `score.json`.
- Reset button to start a new game at any time.
- Theme button switches between light and dark colours.
- Best score ever achieved is displayed beside the current score.
- Help button shows a popup explaining the controls.
- Game over popup appears when no more moves are possible.

## Running

Ensure you have Python 3 and Tkinter installed. Run the game with:

```bash
python3 main.py
```

Use your arrow keys to play. Enjoy!
